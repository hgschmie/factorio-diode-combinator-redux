# Diode Combinator (Redux)

This is an improved version of the [Signal Diode combinator](https://mods.factorio.com/mod/diode-combinator) by [utoxin](https://mods.factorio.com/user/utoxin).

- supports inverting the output signals
- supports input selection
- Custom GUI

## Mod support

- [CompactCircuit](https://mods.factorio.com/mod/compaktcircuit) (filter combinators can be used in compact circuits)
- [Picker Dollies](https://mods.factorio.com/mod/PickerDollies) (filter combinators can be moved and rotated)
